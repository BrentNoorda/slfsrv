Example slfsrv applications and commands

EXAMPLE APPLICATIONS BUILT WITH HTML/JS/CSS USING THE BROWSER FOR UI:

A few examples built with slfsrv:

  • hello-world - very minimal example
  • globals - display the values in the SLFSRV global
  • cmd-shell - a simple command-line tool using your browser as the UI to demonstrate the SLFSRV.exec() method (with command-line completion)
  • menu - demonstrate how a single menu in a browser window can link both to the web and to your local programs and file system - (NOTE: this is one of my favorite uses of slfsrv, but this particular example needs more work because many of the examples in the menu will be particular to my own system and need some time to clean it up)

Examples to test individual slfsrv features:

  • store-test - a short test of the SLFSRV.store() calls, which allow permanent data store on your local computer
  • file-and-dir-test - a short test of the SLFSRV.dir... and SLFSRC.file... calls, which allow reading and writing of your local file system
  • tempdir-test - a short test of the SLFSRV.tempdir, which both creates temporary directories and can initialize them with data from bundled or compiled slfsrv sources
  • Library can be used as nested onsuccess/onerror callbacks or using Javascript Promises (.then().then().then()...)
    • directory listing callback-version - run pyramid-style using onsuccess/onerror callbacks
    • directory listing promises-version - run Javascript Promises version of all calls

EXAMPLE COMMAND OPTIONS
